# AI-Powered-Career-Coach-For-Professoinal-Growth-and-Success
AI-Powered Career Coach is a full-stack web application that uses **Generative AI, AI, and NLP** to support resume analysis, ATS scoring, career path recommendations, and interview preparation. It provides personalized career guidance by combining learning, assessment, and modern web technologies.

## Technologies Used
- React / Next.js
- Node.js / Express
- Python (AI, ML & NLP)
- Supabase (Database & Authentication)

## Features
- AI-based resume analysis with ATS scoring and skill gap identification
- Personalized career path recommendations with structured learning roadmaps
- Integrated learning resources, quizzes, and daily programming challenges
- AI chatbot (Career Mentor) for interview preparation and career guidance
- Progress tracking dashboard with achievements, badges, and analytics
